## 🏃 How to run it

### Clone the repo
```bash
  git clone git@github.com:EricksonSiqueira/mp-tela-login-tailwind.git
```

### Enter the project folder
```bash
  cd mp-tela-login-tailwind
```

### Install all the dependencies
```bash
  npm i
```

### Run the server
```bash
  npm run dev
```
